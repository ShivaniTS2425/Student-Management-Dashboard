Student Management Dashboard

A responsive and interactive Student Management System built using HTML, CSS, and JavaScript. This web application allows users to manage student records efficiently with a clean dark-themed user interface and persistent data storage using the browser's localStorage.

Features

* Add new student records

* Edit existing student details

* Delete student records

* Search students by Register Number, Name, Email, or Course

* Data stored in localStorage (persists after page refresh)

* Modern dark theme UI

* Responsive layout

Technologies Used

 * HTML5

 * CSS3

 * JavaScript (Vanilla JS)

 * Browser LocalStorage
